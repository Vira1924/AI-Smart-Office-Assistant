# AI Smart Office Assistant

## Deskripsi Project
AI Smart Office Assistant merupakan project automation berbasis shell scripting yang digunakan untuk membantu monitoring perangkat kantor secara otomatis. Sistem ini dirancang untuk mempermudah administrator IT dalam memantau kondisi perangkat seperti penggunaan CPU, penggunaan memori, dan koneksi internet secara realtime.

Project ini mengimplementasikan konsep automation system menggunakan Bash Shell pada Linux dan mendukung integrasi Artificial Intelligence (AI) untuk membantu analisis kondisi sistem kantor.

---

# Fitur Utama
- Monitoring penggunaan CPU
- Monitoring penggunaan RAM
- Pengecekan koneksi internet
- Penyimpanan log monitoring otomatis
- Automation system menggunakan shell scripting
- Dukungan integrasi AI untuk analisis sistem
